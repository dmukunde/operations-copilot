# 🚀 Operations Copilot

AI-powered operations and workload management assistant for logistics and operations teams.

Operations Copilot is a prototype AI application that helps operations professionals prioritize work, organize complex workloads, manage focus time, and make faster operational decisions.

The application simulates how an AI copilot could integrate with enterprise tools such as Outlook, Teams, CargoWise, transportation management systems, and airline platforms to support daily operations.

# Application Overview

## Morning Brief

Provides an executive summary of operational priorities, critical blockers, shipment risks, customs issues, meetings, and recommended first actions.

## Smart Inbox

Filters operational communications to surface only emails requiring immediate attention.

## AI Focus Planner

Builds an optimized work schedule by balancing urgency, business impact, deadlines, dependencies, and available focus time.

## Break Intelligence

Encourages sustainable productivity through intelligent focus sessions, work rhythm recommendations, and break reminders.

## Operations Assistant

Generates operational guidance, recommended actions, customer communication drafts, and suggested next steps for selected tasks.

## Productivity Analytics

Visualizes workload distribution, focus allocation, and operational capacity throughout the day.

# Running the Application

1. Clone the repository

```
git clone https://github.com/dmukunde/operations-copilot.git
```

2. Open the project folder.

3. Launch `index.html`

or

```
python -m http.server 8080
```

Then open:

```
http://localhost:8080
```
